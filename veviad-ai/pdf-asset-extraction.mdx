## Purpose

AI-assisted asset creation reduces manual entry from invoices, delivery notes, and equipment documents. Veviad prepares reviewable asset candidates; it does not treat extracted data as an automatic source of truth.

![Review AI-assisted asset creation from a PDF](/images/screenshots/pdf-asset-intake.webp)

## Before you begin

You need Asset create and AI-import permissions, an active AI/OCR integration, and a supported PDF without secrets unrelated to the asset.

## Create assets from a PDF

1. Open the Veviad AI PDF import.
2. Upload the PDF.
3. Wait for OCR and data extraction to finish.
4. Review every suggested asset and field.
5. Correct category, status, tag, serial number, cost, dates, supplier, manufacturer, and custom fields.
6. Resolve required-field validation.
7. Confirm only the reviewed assets.
8. Open each created asset and verify its attachments and values.

## Expected result

Veviad creates only the assets you confirmed and retains the processing status for troubleshooting.

## Best practices

- Split unrelated documents into separate jobs.
- Redact bank data, personal data, and credentials before upload.
- Never accept a guessed serial number or cost without source evidence.
- Use bulk import instead when the source data is already structured.

## Troubleshooting

Check the file type and size, OCR provider health, processing status, queue health, and required fields. A failed extraction does not create an asset.
