Veviad supports AI-assisted extraction of asset data from PDF invoices and similar documents. This workflow is intended to reduce manual entry while keeping a review step in place.

![The assets list with toolbar actions used to start Veviad AI Import](/images/screenshots/assets.webp)

_View shown: `demo.veviad.cloud/assets`_

## Workflow at a glance

1. upload a supported PDF document
2. allow the extraction process to read and classify the content
3. review extracted values in the guided interface
4. correct or enrich the results where needed
5. create the resulting asset records

## What makes the workflow useful

- faster intake from supplier documentation
- reduced copy-and-paste work
- reviewable results before records are created
- alignment with supplier and manufacturer data already stored in Veviad

## Operational note

Extraction quality can vary by document quality, OCR needs, and data consistency. Treat the workflow as an accelerator, not a substitute for final review.
