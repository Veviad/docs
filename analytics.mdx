# Run the numbers, trace the events.

Analytics is the read-only lens over the workspace. Reports surface configurable views of assets, licenses, users, and activity. Audit logs record who did what, when, and to which entity.

## Highlights

- Six built-in report templates plus scheduled custom reports
- Audit logs filtered by user action, security event, or system event
- KPI tiles for executions, failed runs, and active schedules
