Release snapshots record which application baseline the handbook and operator runbooks were checked against. They are documentation evidence, not a replacement for GitHub releases, image digests, or deployment records.

## Evidence rule

A production release record must still include the exact Git tag, GitHub release, image digest, migrations, health result, and rollback decision.
