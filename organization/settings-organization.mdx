Organization settings define the shared structure used by every record.

## Before you begin

You need the relevant Settings permission. Changes affect all users.

## Configure the organization

1. Open **Settings**.
2. Set currency and display preferences under **General**.
3. Maintain locations, departments, and cost centers.
4. Maintain suppliers and manufacturers used by assets and maintenance.
5. Save and confirm the new value in a representative record.

## Expected result

New and edited records use the current organizational options. Existing links remain intact when a value is renamed.

## Best practices

- Prefer deactivation over deletion when a value is already in use.
- Use stable cost-center numbers and clear plant/location names.
- Assign managers through the People model, not application-access records.

## Troubleshooting

A missing Settings page usually means the user lacks its granular permission.
