Release-Snapshots halten fest, gegen welchen Anwendungsstand Handbuch und Operator-Runbooks geprüft wurden. Sie ersetzen weder GitHub Release noch Image-Digest oder Deployment-Nachweis.

## Aktueller Snapshot

Der [aktuelle Release-Snapshot](/de/release-notes/current) nennt Quell-SHA, Laufzeitbasis sowie sichtbare und betriebliche Änderungen dieses Handbuch-Refreshs.

## Nachweisregel

Ein Produktionsrelease benötigt weiterhin exakten Git-Tag, GitHub Release, Image-Digest, Migrationen, Health-Ergebnis und Rollback-Entscheidung.
