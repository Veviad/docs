## Richtigen Bereich wählen

| Verantwortung | Konfiguration |
| --- | --- |
| Gemeinsame Organisationsdaten | Allgemein, Standorte, Abteilungen, Kostenstellen, Lieferanten, Hersteller |
| Asset-Lebenszyklus | Status, Kategorien, Abschreibung, Tags, Etiketten, Zusatzfelder, Feldregeln |
| Arbeitsausführung | Checklisten, Wartungstypen, Actions, Benachrichtigungen, Übergaben |
| Benutzerpräferenzen | Profilsprache, Zeitzone und Benachrichtigungen |
| Anmeldung und Provisionierung | Benutzer, Einladungen, Rollen, Integrationen, SSO und SCIM |

## Berechtigungsverhalten

Einstellungen verwendet granulare Rechte. Ein Benutzer kann einen Bereich verwalten, ohne alle Einstellungsseiten zu sehen.

## Bewährte Vorgehensweisen

- Richten Sie gemeinsame Werte vor Importen ein.
- Deaktivieren Sie referenzierte Werte.
- Testen Sie mit einem repräsentativen Datensatz.
- Prüfen Sie nach wesentlichen Änderungen das Audit-Protokoll.
