## Zweck

Das Dashboard ist ein persönlicher Arbeitsbereich und keine starre Unternehmenskennzahlenseite.

![Maschinenbau-Dashboard mit Widgets](/images/screenshots/de/dashboard.webp)

## Voraussetzungen

Sie benötigen Dashboard-Zugriff und für analytische Widgets zusätzlich Zugriff auf die verwendete Datendomäne.

## Dashboard anpassen

1. Öffnen Sie **Dashboard**.
2. Wählen Sie **Dashboard bearbeiten**.
3. Fügen Sie ein Standard-Widget hinzu oder öffnen Sie den Widget-Builder.
4. Wählen Sie Domäne, Visualisierung, Kennzahl, Gruppierung, Filter und Titel.
5. Prüfen Sie die Vorschau.
6. Ordnen und skalieren Sie unterstützte Widgets.
7. Speichern Sie das Layout.

![Bearbeitungsmodus des Dashboards](/images/screenshots/de/dashboard-edit.webp)

![Konfigurierbarer Widget-Builder](/images/screenshots/de/dashboard-widget-builder.webp)

## Erwartetes Ergebnis

Das Layout ist persönlich. Widgets laden unabhängig, sodass ein fehlerhaftes Widget den restlichen Arbeitsbereich nicht blockiert.

## Bewährte Vorgehensweisen

- Begrenzen Sie die erste Reihe auf täglich entscheidungsrelevante Werte.
- Nutzen Sie Berichte für Detailanalyse.
- Benennen Sie Widgets als Frage, etwa „Maschinen mit fälligem Service“.
- Setzen Sie das Layout bei Bedarf auf Standard zurück.

## Fehlerbehebung

Ein leeres Widget kann keine Treffer, ein entferntes Feld oder eine fehlende Domänenberechtigung bedeuten.
