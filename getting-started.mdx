Use this section when you are opening Veviad for the first time or preparing a new workspace for your team. It explains how to sign in, move through the interface, and set up the first pieces that make the rest of the product easier to use.

## Recommended first path

1. Confirm access and sign in.
2. Learn the main navigation and global search.
3. Review the workspace defaults.
4. Add the first organizational data such as departments, locations, and categories.
5. Create or import the first assets.
6. Review assignment and handover workflows before rolling out to more users.
