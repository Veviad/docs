# Common Questions

## Where should a new user start?

Start with [Getting Started](/getting-started), then move into the module that matches the user's daily work.

## Where do I configure shared structure such as departments or locations?

Use [Settings](/organization/settings). Shared taxonomy and system-wide rules belong there rather than in individual records.

## How do I find something quickly?

Use the in-app global search shortcut:

- `Cmd+K` on macOS
- `Ctrl+K` on Windows and Linux

## What should I use for repeatable operational processes?

Use [Checklists and Workflows](/main/workflows): checklists for visible step-by-step execution, and workflow actions when lifecycle events should trigger follow-up automatically.

## How do invitations usually fail?

The most common causes are:

- wrong email address
- email delivered to spam or junk
- expired or revoked invitation

Use [Admin](/organization/admin) for managing invitations.

## Where do I check whether the environment is healthy?

Use the [Admin Dashboard](/organization/admin) to review system health before contacting support.

## Is Veviad AI a replacement for manual review?

No. Veviad AI is best used as an accelerator for discovery, analysis, and intake. For sensitive operational or compliance decisions, review the underlying records.

## Where can I ask for help?

- this docs site for the public manual
- `support.veviad.com` for support workflows
- `support@veviad.com` for direct support contact
