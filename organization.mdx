# Configure the workspace around the work.

Organization & Administration covers the tenant-level setup that workspace administrators use to keep Veviad useful for everyone: shared taxonomy, workspace rules, authentication, permissions, invitations, and integrations. It belongs in the user manual because these are product-administration tasks, not internal infrastructure or release operations.

## Highlights

- Settings for currency, depreciation, taxonomy, asset tags, handover, notifications, actions, and checklists
- Admin tools for roles, users, invitations, integrations, and system health
- SSO, SCIM provisioning, and group-to-role synchronization for managed identity rollouts
