This public docs launch focuses on the operational and administrative product experience rather than internal engineering detail.

## Included in the current baseline

- getting started guidance
- core module overviews for dashboard, assets, people, supporting inventory, and reports
- workflow documentation for checklists, audit logs, invitations, and automation
- asset maintenance documentation for service history, PDF reports / invoices, maintenance types, and scheduled maintenance actions
- administration guidance for settings, notifications, SSO, and system info
- Veviad AI and PDF extraction overviews
- FAQ coverage for common rollout and support questions

## Intentionally not included

- public API documentation
- deployment runbooks
- internal testing notes
- engineering completion trackers and ETA language

## Next likely growth areas

- expanded German content
- deeper task-based walkthroughs
- future semantic search enhancements when the docs corpus grows
