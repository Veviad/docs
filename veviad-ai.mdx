Veviad AI provides three controlled assistance paths:

- [Assistant](/veviad-ai/assistant) for natural-language questions about permitted workspace data.
- [Report AI summaries](/analytics/reports) for a grounded interpretation of a completed report run.
- [Create assets from PDFs with AI](/veviad-ai/pdf-asset-extraction) to prepare candidate asset fields for review before creation.

## Safety model

AI uses the current user's authorization scope. Output can be incomplete or incorrect and must be verified against source records. Never provide credentials, secrets, or unrelated personal data in prompts.

## Availability

The feature appears only when the required integration is active and the user has permission. A provider can be configured but unhealthy; administrators should review integration and operator health separately.
